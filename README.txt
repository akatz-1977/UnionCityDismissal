Student Pickup Number App

Features:
- Numbers only
- Up to 10 active numbers
- Numbers stay until Done is tapped
- Recall and Clear All controls
- Full-screen display for AirPlay/projector
- Saves on the device and works offline after first load

Publish:
Upload the individual files to a new GitHub repository root.
Settings > Pages > Deploy from a branch > main > /(root).

Important:
This version syncs between tabs/windows on the SAME device/browser.
It is ideal for one staff member using Display Mode and AirPlay mirroring.
Separate-device live syncing would require Firebase or Supabase.
